== Description ==

A simple plugin to set purchasable quantity multiples for WooCommerce products. Allow to set a specific multiple for the quantity of products that can be added to the cart. For example, if a multiple of 3 is set, users can add products in quantities of 3, 6, 9, etc.
It adds a field on woocomerce product edit page and allow you to set purchasable quantity multiple

== Installation ==

1. Upload `quantity-multiples` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Usage ==

This plugin adds a field on woocomerce product edit page to set purchasable quantity multiple
